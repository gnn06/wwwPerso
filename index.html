Qui suis-je ?

Bonjour,

Je suis chef de projet informatique chez Docapost-BPO à sophia-antipolis depuis une dizaine d'années. Je travaille dans le développement web J2EE et j'encadre l'équipe gérant le site www.infogreffe.fr

Pas grand chose sur ce site à part quelques liens vers des outils que j'utilise.

Mes Développements

J'ai développé une extension pour thunderbird Tasks & mails qui permet de gérer des tâches avec la possibilité de les lier aux emails ce qui m'évite de constamment jongler entre Thunderbird et un gestionnaire de tâches.
J'ai entamé le développement d'une extension Folder Plus pour Thunderbird afin de rajouter quelques fonctionnalités manquantes autour des folders.

Mon github.

Toujours interessé par les dév web, j'ai testé les outils suivants : les CMS SPIP, Drupal et CouchCMS ; le langage de CSS LESS ; le générateur de site statique Jekyll et le framework CSS KNACSS.

J'ai utilisé le framework AngularsJS pour réaliser une extension Chrome pour extraire les données d'une page web. AngularJS est très sophistiqué mais nécessite trop, à mes yeux, de rentrer dans le framework en debug pour le comprendre, le maitriser. De plus, le framework manque de stabilité dans le temps.

Angular JS Material m'a permis de réaliser PhotosCloud, une webapp de gestion de photos avec un backend NodeJS hébergé sur mon raspberry pi.

Je trouve Python intéressant. NodeJS est un écosystème que je trouve particulièrement prometteur. Ruby, certes puissant ne me plait pas trop.

J'utilise le très intéressant Microsoft Visual Studio Code.

Le développement d'application Android est un mal nécessaire :-) mais je n'arrive pas à adhérer à la philosophie.

Un lien vers mes albums privés Google Photos.

Un autre centre d'intêret : la 3D. Pour la réalisation rapide de plan, j'utilise Google sketchup.

La partie mécanique

Mon principal hobby : l'automobile. Je roule actuellement en clio rs3. Au préalable, j'ai roulé en Twingo RS, Yaris TS et plus longuement en Clio RS

Je fais un peu de circuit en loisir avec le japan power. Je me suis notamment fait plaisir pendant deux ans avec une Honda CRX. La caisse avait été acheté avec mon co-équipier Le ciao. Pour le chronométrage, je conseille race chrono.

Pour les amateurs de Renault Sport présents dans le 06, je conseille le forum Renault Sport Passion 06.

Pour mes trajets boulot-dodo et aller à la plage le week-end, j'ai basculé dans le deux roues avec d'abord, un scooter 125 xmax 125 puis une Honda CBF 600 et maintenant une BMW F700GS.

Pour mes trajets entre Nice Nord et Sophia-antipolis, j'ai testé le covoiturate et la ligne 230 de bus entre Nice et Sophia.
